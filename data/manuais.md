# Guia de Suporte Técnico - ERP & Redes

## Switch Cisco Nexus
- Para configurar o IP da gerência, use o comando: `interface mgmt0` seguido de `ip address [IP]/[MASCARA]`.
- O acesso inicial deve ser feito via porta console.

## Roteador MikroTik
- O acesso padrão é feito via Winbox na porta 8291.
- Para habilitar a API de automação, use: `/ip service enable api`.
